# Raycast

Personal collection of Raycast extensions, scripts, snippets, and assets.

## Structure

- `extensions/` - Custom Raycast extensions
- `scripts/` - Script commands for Raycast
- `snippets/` - Text snippets (exported for backup/version control)
- `assets/` - Shared resources (icons, images, templates)

## Setup

1. Clone this repository
2. Import extensions via Raycast preferences
3. Add script commands directory in Raycast settings
4. Sync across devices by pulling latest changes

## Syncing Between Devices

```bash
# Pull latest changes
git pull

# Push your changes
git add .
git commit -m "Update extensions/scripts"
git push
```
