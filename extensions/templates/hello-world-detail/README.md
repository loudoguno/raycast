# Hello World - Detail Template

A simple Raycast extension template using the Detail view to display markdown content.

## Features

- Displays formatted markdown content
- Simple structure perfect for getting started
- Easy to customize and extend

## Getting Started

1. Copy this template to your extensions directory
2. Run `npm install && npm run dev`
3. Open Raycast and search for "Hello World"
4. Edit `src/index.tsx` to customize the content

## Structure

```
hello-world-detail/
├── src/
│   └── index.tsx       # Main command
├── package.json        # Extension metadata
└── README.md          # This file
```
