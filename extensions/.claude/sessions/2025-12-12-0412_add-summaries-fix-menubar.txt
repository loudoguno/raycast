‚è∫ Done! The extension is now running. Here's what I changed:                    
                                                                                
  Problem: The menu-bar command runs every 10 minutes (via the interval setting)
   and the AppleScript was:                                                     
  1. Activating Safari if it wasn't running                                     
  2. Creating a new tab if no usage tab existed                                 
                                                                                
  This caused Safari windows to pop up unexpectedly.                            
                                                                                
  Solution: I created two modes for the AppleScript:                            
                                                                                
  1. Quiet mode (used by background/interval refreshes):                        
    - Checks if Safari is already running using System Events                   
    - Only reads from an existing Claude usage tab                              
    - Never opens Safari or creates new tabs                                    
    - Returns gracefully with null data if Safari isn't open or tab doesn't     
  exist                                                                         
  2. Interactive mode (used when user clicks "Refresh"):                        
    - Works like before - opens Safari and creates tab if needed                
                                                                                
  Changes made:                                                                 
  - src/utils/applescript.ts: Added APPLESCRIPT_QUIET and renamed original to   
  APPLESCRIPT_INTERACTIVE. Added quiet option parameter to runAppleScript().    
  - src/menu-bar.tsx: Updated fetchUsage() to use quiet mode by default,        
  interactive mode only when user manually refreshes.                           
                                                                                
  The menu bar will now show C: ‚Äî if Safari isn't running or no usage tab       
  exists, but won't pop up windows. Users can manually refresh (Cmd+R) to open  
  Safari if needed.
