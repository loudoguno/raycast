## PROJECT GOAL:

Build custom Raycast extensions for SalesSprint to streamline workflows and improve productivity.

## INITIAL REQUIREMENTS:

- Repository for developing and maintaining multiple SalesSprint-specific Raycast extensions
- Follow Raycast extension development best practices
- Use TypeScript for type safety and better developer experience
- Implement proper error handling and user feedback
- Support for SalesSprint's internal tools and workflows

- Each extension should be self-contained with its own package.json and dependencies
- Extensions should be tested thoroughly in Raycast's developer mode before publishing
- Follow SalesSprint's branding guidelines for icons and naming
- Consider user privacy and data security when integrating with SalesSprint APIs
